# NeuroPath-Godot

Welcome to the NeuroPath-Godot repository! This project integrates reinforcement learning with the Godot Engine, providing an environment for training and testing intelligent agents.

## Project Structure

- `env_godot/`: The Godot 4.1 project containing the simulated environment, scenes, and assets.

## Requirements

- **Godot Engine:** Version 4.1 or compatible.

## Getting Started

1. **Environment Setup:**
   Open the Godot Editor (v4.1) and import the project located in the `env_godot/` folder. The main scene is typically set to `res://Scenes/world.tscn`.
